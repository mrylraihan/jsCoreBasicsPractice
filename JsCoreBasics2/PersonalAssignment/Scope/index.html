Scope Explained in Javascript

  • Global Scope
    • global variables can be accessed from anywhere
    • in the context of a browser, they are found on the window object (aside from let and const)
  • Function Scope
    • Variables declared within a function can only be accessed within that function
    • the older var keyword used function scope
  • Block Scope
    • the newest form of scope, variables declared with in a {} cannot be accessed outside of it
    • made possible with let and const .

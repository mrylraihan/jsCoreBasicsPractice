Arguments vs Parameters

"parameters" and "arguments" interchangeably.
Technically, there is a difference though:

Parameters are these variables which you specify between parentheses when defining a function.

function sayHi(name) { ... }

In this example, name is a parameter

Argumentsthen are the concrete values you pass to a function when calling that function:

sayHi('Max');

'Max' is an argument of the function therefore - for the name parameter to be precise.
